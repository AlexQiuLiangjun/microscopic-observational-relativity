# 微观观测相对论 2.0 核心摘要

理论主名：微观观测相对论（Microscopic Observational Relativity, MOR）

副标题：基于多维投影结构的量子实在性重构

2.0 定位：弱版本，哲学表达增强版。MOR 是量子力学的本体论诠释，保留标准量子力学的计算结果和 Born 规则，目标是重解释叠加、坍缩、观测、概率与实在性。

核心命题：量子态的叠加不是现实缺席，也不必诉诸宇宙分裂，而是表示同一微观实体在多个相互关联的投影扇区中具有共同实在性。观测不是创造结果，而是在物理测量关系中读取某一投影显现。

核心公式：

```text
M = (O, H_int, E, D)
Ω_Q = ({S_i}, {ψ_i}, Γ, μ)
μ_i(M) = ||Π_i^M |ψ⟩||²
```

强版本接口：未来若要提出新物理预测，必须引入受严格约束的 δ_i^MOR；这不是 2.0 弱版本的核心主张。
