# 微观观测相对论 5.0 强版本启动摘要

定位：5.0 是强版本启动稿，不声称新物理已经成立。它把多维投影结构从解释语言提升为潜在物理结构，并要求其在特定实验条件下给出区别于标准量子力学的可检验偏离。

核心变量：

```text
P_i^MOR(M) = Tr(Π_i^M ρ) + δ_i^MOR(M, Ω_Q, χ_Q, D, C)
Σ_i δ_i^MOR = 0
δ_i^MOR → 0 in classical/decohered limits
```

强版本硬约束：归一化、非负性、无信号原则、经典极限回收、与普通噪声/退相干可区分。

第一候选效应：投影曲率与切面滞后。若系统经历 A-B-A 顺序不相容测量，在控制普通量子通道、仪器记忆和退相干后，仍出现依赖测量基切换路径的残差 H_ABA，则可能成为 MOR 强版本候选信号。

第二候选效应：动态高阶干涉。普通静态线性三缝不应偏离标准量子力学；但在相干保持、弱扰动、动态切面切换条件下，可寻找小的 Sorkin 型参数 κ_MOR。

优先实验路线：A-B-A 顺序测量、动态三缝/多路径干涉、弱测量与前后选择、扩展 Wigner 朋友、介观物质波干涉。

最重要的反向问题：δ_i^MOR 最可能表现为概率偏移、相位漂移、额外退相干，还是测量历史相关的滞后效应？
