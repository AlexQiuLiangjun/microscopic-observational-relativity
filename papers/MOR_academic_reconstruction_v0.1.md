## 摘要

本文将“微观观测相对论”（Microscopic Observational Relativity, MOR）重新定位为一种量子力学本体论诠释框架，而非替代标准量子力学的动力学理论。本文承认 MOR 与关系量子力学（RQM）、退相干理论、量子达尔文主义以及量子参照系（QRF）文献存在显著重叠；其目标不是宣称全新物理理论，而是在这些已有路线的语境中提出三个可讨论的增量：未显现域的正面本体论地位、观测切面与跨切面不变量的系统区分、以及微观关系域先于宏观空间的涌现性承诺。

本文的核心主张是：量子测量不应被理解为完整实在的物理断裂，而应被理解为某一观测参照系中显现切面的事实化。相对事实不是主观知识状态；公共事实也不是无参照系的全维实在，而是经退相干、环境冗余记录和公共投影面稳定化后的事实层。本文修正早期 MOR 文稿中若干仅具启发意义的符号表达，并给出更稳妥的形式化草案：以密度算符、可观测量代数限制、Lüders 规则和信息损失刻画观测切面与未显现域。

本文仍属于概念性弱版本。它不推导 Born 规则，不提出新实验偏离，也不声称解决所有测量问题。它的贡献在于给出一种关系实在论与显现本体论结合的表述方式，并为后续与 RQM、Stable Facts/Relative Facts、Wigner 的朋友、QRF 与时空涌现文献的系统对话建立可引用草稿。

## 一、问题与定位

量子力学的测量问题长期困扰着物理学和哲学：如果量子态按照线性规律演化，为什么测量中会出现单一确定结果？如果波函数坍缩只是计算规则，那么它对应怎样的本体论图像？如果事实依赖观测参照系，那么客观世界又如何稳定出现？

MOR 的早期表述试图用“观测切面”“未显现域”“微观关系域”和“公共投影面”等概念回答这些问题。但严格审视后必须承认：其中许多直觉与已有文献高度相关，尤其是 Rovelli 的关系量子力学、Di Biagio 与 Rovelli 对相对事实和稳定事实的区分、Zurek 的退相干与量子达尔文主义、以及近年的量子参照系研究。

因此，本稿不再以“提出全新理论”为定位，而是把 MOR 改写为一个本体论重构方案：在 RQM 与退相干/量子达尔文主义框架附近，突出未显现域、切面不变量和空间涌现三个增量。

> 本文的核心问题不是“MOR 是否完全新颖”，而是“MOR 在已有关系性量子诠释中是否提出了有意义的本体论增量”。

## 二、本文的原初直觉

MOR 的原初直觉并非来自对某一篇文献的推演，而来自一个朴素但持续的疑问：我们是否把宏观经验中的“物体、空间、属性、观测”过早投射到了微观世界？在宏观经验中，我们总是从某一角度、某一通道、某一相互作用方式读取对象；我们得到的是对象的一个切面，而不是对象的全维实在。

这个直觉在量子问题中被进一步放大。微观系统不应被想象为带有完整经典属性的小球。更合适的图像是：系统由希尔伯特空间中的态结构、投影关系和相互作用通道描述；观测不是看见对象本身，而是在某一参照系下取得一个可记录的投影切面。

因此，MOR 最想保留的思想锋芒是：没有本体论意义上的坍缩，只有切面事实化；没有无参照系的完整客观现实，只有稳定共享的公共投影面；空间未必是底层舞台，空间可能是微观关系域在宏观显现层中的涌现结果。

> 在 MOR 的本体论图像中，微观底层不应首先被理解为彼此孤立的实体，而应被理解为关系结构；测量不被解释为全局实在的坍缩或断裂，而是某一观测切面的事实化；宏观客观世界也不是全维实在的无参照系呈现，而是由稳定记录、退相干和共同相互作用通道形成的公共投影面。

## 三、与既有文献的关系

### 3.1 关系量子力学：相对事实的核心邻近性

RQM 的核心思想是，量子状态和事实总是相对于物理系统而言的，而非相对于一个无参照系的全局观察者。MOR 的“观测切面”与这一思想高度接近：事实不是主观心理状态，而是在物理相互作用中相对于某一系统或观测参照系显现的事实。

MOR 与 RQM 的差异不应被夸大。更合适的说法是：MOR 接受 RQM 的相对事实方向，并试图补充“未事实化结构”的正面本体论地位，即未显现域。MOR 的问题是：当某个切面成为事实时，未在该参照系中显现的结构应如何被理解？它既不能被还原为主观无知，也不能被说成局域隐藏变量。

### 3.2 稳定事实、退相干与公共投影面

Di Biagio 与 Rovelli 的 Stable Facts/Relative Facts 区分，与 MOR 的“观测切面/公共投影面”高度相关。MOR 应承认：公共投影面不是凭空提出，而是与相对事实经退相干稳定化成为稳定事实的思路同源。

Zurek 的退相干、环境诱导超选择和量子达尔文主义进一步说明，环境可以冗余记录某些稳定信息，使多个观察者从环境不同片段中读取一致事实。MOR 对此的定位是：退相干与量子达尔文主义解释公共投影面的物理稳定机制；MOR 解释公共投影面为什么不应被误认为无参照系的完整实在。

### 3.3 量子参照系与“相对论”名称的挑战

“相对论”一词在物理学中通常意味着参照系变换与不变量结构。早期 MOR 使用该名称主要是哲学类比，尚未给出严格的观测参照系变换群。因此，若 MOR 要在学术语境中保留“相对论”之名，最现实的方向是接入量子参照系（QRF）文献。

Giacomini、Castro-Ruiz 与 Brukner 的量子参照系工作表明，叠加、纠缠和物理定律形式在不同量子参照系之间可以被严格讨论。对 MOR 而言，QRF 提供了把“观测切面相对性”数学化的现成入口。

## 四、MOR 的三个拟议增量贡献

### 4.1 未显现域：未事实化结构的正面本体论地位

MOR 最重要的增量候选是未显现域。未显现域在早期草稿中曾称为“遮蔽域”，但“未显现域”更能避免被误读为某种已经完成、只是被挡住的经典对象。本文所称未显现域，不是观察者不知道的既定经典属性集合，也不是被测量摧毁的世界，而是当前观测参照系下未取得事实显现地位的关系结构。

这个概念的风险很大：如果未显现域被解释成“所有可能结果都预先以经典方式存在，只是没有被观察”，它会退化为局域隐藏变量图像，并与 Bell/Kochen-Specker 方向冲突。MOR 因此必须坚持：未显现域是关系性和非经典的未事实化结构，而不是经典预存答案库。

### 4.2 切面与不变量：类比相对论精神的系统区分

MOR 的第二个增量候选，是把量子系统的属性分为切面依赖量与跨切面不变量。某方向自旋值、一次测量中的位置记录、宏观轨迹等，多数属于观测切面；而电荷、静质量、自旋模长、守恒量、群表示不变量与统计类型等，更接近跨切面的结构性不变量。

> 切面会变，不变量不变。经典属性多半属于切面；固有属性属于跨切面的结构。

这一点是 MOR 最接近“相对论”精神的地方。正如狭义相对论区分坐标依赖量与洛伦兹不变量，MOR 可尝试区分观测参照系依赖的事实切面与跨参照系保持的结构不变量。

### 4.3 微观关系域先于空间

MOR 的第三个增量候选，是微观关系域先于宏观空间的涌现性承诺。MOR 不把微观对象理解为小球在预先给定空间中运动，而把量子系统理解为关联、相互作用、可显现性、因果约束与粗粒化规则构成的关系域。

```text
R_Q = (A_Q, I_Q, P_Q, C_Q, K_Q)
```

其中 A_Q 表示关联结构，I_Q 表示相互作用通道，P_Q 表示可投影/可显现结构，C_Q 表示因果可达性与无信号约束，K_Q 表示从微观关系域到宏观空间显现的粗粒化规则。这个表达式只是概念坐标，不是成熟物理方程。

## 五、修正后的形式化草案

早期 MOR 使用 Ω_Q、Σ_M 与 Λ_M 等记号，其中部分表达只是概念示意。学术化版本必须避免把未定义符号伪装成数学。本文采用更保守的形式化草案。

### 5.1 状态与观测参照系

令量子系统 Q 由希尔伯特空间 H 上的密度算符 ρ 表示。观测参照系 M 可近似表示为一组可共同测量的可观测量，或一个交换子代数 A_M；在理想投影测量情形下，也可表示为一组投影算符 {P_i^M}。

```text
ρ ∈ D(H),    M ↦ A_M 或 {P_i^M}
```

### 5.2 观测切面

在代数语言下，观测切面可理解为态 ρ 在可观测量代数 A_M 上的限制。它不是全态 ρ 本身，而是当前观测参照系可访问、可记录、可比较的事实结构。

```text
Σ_M(ρ) := ρ|_{A_M}
```

在理想投影测量中，结果 i 的概率仍由 Born 规则给出：

```text
p_i = Tr(P_i^M ρ)
```

若结果 i 已被记录，条件态应按 Lüders 规则写作：

```text
ρ_i' = P_i^M ρ P_i^M / Tr(P_i^M ρ)
```

对于复合系统 AB，在 A 侧局部测量时，应写作：

```text
ρ_a' = (P_a^A ⊗ I_B) ρ_AB (P_a^A ⊗ I_B) / Tr[(P_a^A ⊗ I_B) ρ_AB]
```

这修正了早期文稿中只写左乘投影算符的错误。局部测量不是 A 向 B 发送信号，而是在非可分联合态中条件化出一个相关切面。

### 5.3 未显现域

早期表达 Λ_M(Q) = Ω_Q − Σ_M(Q) 只能作为概念示意，不能作为数学公式。更稳妥的说法是：未显现域表示从全态 ρ 到参照系可访问结构 Σ_M(ρ) 的限制、粗粒化或去相干过程中未被保留的结构。

一种候选刻画方式是引入与 M 相关的去相干/条件期望映射 D_M，并用 ρ 与 D_M(ρ) 之间的信息差异来度量未显现程度。例如可探索相对熵、相干性度量或代数限制中的信息损失。

```text
Λ_M：由 ρ → Σ_M(ρ) 或 ρ → D_M(ρ) 产生的信息损失结构
```

本文不把该表达当作完成的数学定义，只把它作为未来形式化的方向。

## 六、量子纠缠的 MOR 重构

量子纠缠是 MOR 的核心检验场。MOR 不把纠缠理解为两个空间分离粒子之间的超距通信，而把它理解为联合微观关系域在局部观测切面中的相关显现。

```text
|Ψ_AB⟩ ≠ |ψ_A⟩ ⊗ |ψ_B⟩
```

这意味着纠缠系统的完整信息不能拆成 A 的完整信息加 B 的完整信息。宏观空间把 A 和 B 呈现为两个空间分离的对象；但这种空间分离并不穷尽它们在联合希尔伯特空间中的非可分结构。

> 空间分离不等于关系分离。位置独立不等于本体独立。局部记录不等于完整实在。

测量 A 后，已知 A 的结果可以推断 B 的相关结果，但这不是可控通信。B 的本地统计在不知道 A 的测量结果和设置时仍不可区分。只有通过经典通信比较记录后，联合相关性才成为公共投影面上的实验事实。

Bell 不等式违背在 MOR 中不被解释为超距通信的证明，而被解释为局域经典属性完备性的失败。被否定的是：A 和 B 各自携带一套与测量方式无关的、完备的、局域预存属性。MOR 保留相对论因果律，同时放弃宏观空间分离必然意味着微观关系独立的经典本体论。

## 七、测量问题、单一结果与优先基

MOR 将坍缩解释为切面事实化，但这并不自动解决所有测量问题。至少有两个问题必须公开承认。第一，单一结果问题：为什么事实化的是这个切面而不是那个切面？第二，优先基问题：哪些切面是可事实化的候选切面？

MOR 的弱版本可以采取保守立场：它接受 Born 规则给出的结果权重，接受退相干对指针基和稳定记录的解释，并把“相对事实”视为测量事件中的原始本体论事实。这个立场与 RQM 接近，但 MOR 通过未显现域与切面不变量试图增加解释结构。

若未来发展强版本，MOR 必须解释切面事实化机制、Born 规则来源以及优先基选择，而不能仅用新术语替代旧问题。

## 八、Wigner 的朋友与相对事实冲突

Wigner 的朋友及其扩展 no-go 定理，是所有相对事实理论必须面对的考场。Frauchiger-Renner、Brukner、Bong 等工作表明，观察者无关事实、局域性与量子力学普适性之间存在深刻张力。

MOR 的基本回应方向是：不同观测参照系可以有不同事实切面；公共投影面不是所有切面的简单合并，而是经过交互、退相干、记录比较与跨参照系协调后形成的稳定事实层。但这只是纲领性回答。MOR 必须进一步说明，当朋友的事实切面与 Wigner 的整体实验室切面发生张力时，哪些跨切面链接是允许的，哪些是不允许的。

因此，Wigner 的朋友问题不应被放到未来附录，而应成为 MOR 学术化版本的核心测试之一。

## 九、公共投影面与客观性

公共投影面回答的是：如果事实是相对的，为什么世界仍然客观稳定？MOR 的回答是：宏观客观性不是来自无参照系的全维实在，而是来自大量稳定切面被环境冗余记录、被仪器放大、被观察者共享。

这与量子达尔文主义高度相关。MOR 不应把公共投影面当作原创地重新发明量子达尔文主义，而应承认其物理机制来自退相干与冗余环境记录。MOR 的增量在于：公共投影面即使稳定、共享、可重复，也仍然是显现层，而不是完整微观实在本身。

## 十、空间涌现与虫洞问题

MOR 的微观关系域承诺把问题推向时空本质：宏观空间可能不是微观实在的原始容器，而是微观关系域在稳定观测参照系下的涌现显现。这一点与由纠缠构造时空、ER=EPR 等方向存在启发性联系。

但本文不把虫洞作为解释纠缠的必要假设，也不声称普通纠缠粒子之间存在可穿越虫洞。更稳妥的说法是：虫洞提示了一种可能性，即某些被宏观空间看作远隔的关系，在更深层结构中可能并非真正分离。纠缠首先应被理解为关系非分离性；虫洞只是这种非分离性在引力几何语言中的潜在对应。

| RQM | 事实相对于物理系统或相互作用而成立；MOR 的观测切面与相对事实高度邻近。 | MOR 接受相对事实方向，但尝试给未事实化结构以正面本体论地位，即未显现域；同时提出切面依赖属性与跨切面不变量的区分。边界：不能把 MOR 写成完全独立于 RQM 的新理论。 |
| Stable Facts / Relative Facts | 相对事实可经退相干稳定化为稳定事实；这与 MOR 的公共投影面非常接近。 | MOR 将稳定事实解释为公共投影面，即稳定共享的显现层；其差异点在于强调公共投影面仍不是全维实在本身。边界：必须明确引用并区分 Di Biagio 与 Rovelli 的工作。 |
| 量子达尔文主义 | 环境冗余记录解释经典客观性和多观察者一致性。 | MOR 接受量子达尔文主义作为公共投影面的稳定机制，但补充显现本体论：稳定、共享、可重复不等于无参照系的全维实在。边界：不能把公共投影面当作原创地重新发明量子达尔文主义。 |
| QRF | 量子参照系研究提供严格的参照系变换框架，显示叠加和纠缠可依赖参照系描述。 | QRF 是 MOR 未来数学化“观测参照系变换”的优先入口，也是保留“相对论”命名的最强支撑。边界：当前 v0 尚未真正建立 QRF 形式体系。 |
| QBism | 反对把量子态简单等同为外在客观经典属性。 | MOR 与 QBism 都反对朴素实在论，但 MOR 不把结果还原为主体信念，而保持关系实在论与显现本体论。边界：需防止被误读成主观主义。 |
| Everett | 不把坍缩看作基本动力学断裂，保留整体量子态演化。 | MOR 不把所有分支实体化为平行世界，而强调某一参照系下的切面事实化与公共投影面稳定化。边界：单一结果问题仍需承认。 |
| 玻姆理论 | 同样追求明确本体论，反对纯工具主义。 | MOR 不引入粒子确定轨迹和非局域导引波；其本体论核心是关系域、切面与不变量。边界：不能把未显现域写成玻姆式隐藏变量的弱化版。 |
| GRW/客观坍缩 | 正面面对测量和单一结果问题。 | MOR 不修改薛定谔动力学，也不引入客观坍缩机制；坍缩被解释为切面事实化。边界：MOR 因此不能声称已解决客观坍缩理论试图解决的全部动力学问题。 |

## 十二、边界、风险与后续工作

- MOR v0 不是新物理理论，而是量子本体论重构草案。
- Born 规则尚未推导，只作为经验约束被接受。
- 未显现域必须避免退化为局域隐藏变量集合。
- 切面事实化尚未解释单一结果选择机制。
- 公共投影面与量子达尔文主义高度重叠，必须以差异化定位写清。
- Wigner 的朋友与跨视角事实协调是后续必做内容。
- 若保留“相对论”命名，需要接入 QRF 或其他参照系变换结构。
- 空间涌现与 ER=EPR 只能作为启发性方向，不能写成已成立结论。

## 结论

MOR academic-v0.1 的学术化定位不再是“提出全新的量子理论”，而是“在关系量子力学与退相干/量子达尔文主义的语境中，对相对事实、未显现域和切面不变量做本体论重构”。这个定位更谦逊，也更有机会被认真阅读。

如果 MOR 有独立价值，它不在于重复说明事实相对或环境导致客观性，而在于系统提出三个问题：未事实化结构是否应有正面本体论地位？固有属性是否可被理解为跨切面不变量？宏观空间是否可能是微观关系域在公共投影面中的涌现显现？

> 微观底层首先是关系结构，而非孤立实体；测量首先是切面事实化，而非全局实在断裂；宏观客观世界首先是稳定共享的公共投影面，而非全维实在的无参照系呈现。
