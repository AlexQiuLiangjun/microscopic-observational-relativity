# 微观观测相对论 (MOR) / 多维投影量子论 (MPQT)

**Microscopic Observational Relativity / Multidimensional Projection Quantum Theory**

一个由独立研究者提出的量子力学诠释框架：把量子叠加理解为多维投影结构的真实共存，把观测理解为对"观测参照系"的选择与切面事实化，而非波函数的物理坍缩。本仓库以版本演化的方式完整记录该理论从 2025 年 5 月初稿（MPQT）到后续各版本（MOR）的发展过程。

## 诚实声明（Positioning Statement）

本理论由作者凭物理直觉独立构思，作者并非职业物理学者，写作时未系统研读量子基础（quantum foundations）领域文献。事后比对发现，理论的部分核心直觉与已有研究存在独立重合，包括 Rovelli 的关系量子力学（Relational Quantum Mechanics, 1996）、Zurek 的退相干与量子达尔文主义、以及"由纠缠构造时空"（Van Raamsdonk 2010, ER=EPR 2013）等研究方向。本仓库保留各历史版本原貌，不做追溯性修改，以如实呈现一个业余研究者的思想演化轨迹。

理论的弱版本不声称替代标准量子力学的计算体系，不声称已推导 Born 规则；与已有诠释的严格差异化论证仍在进行中。欢迎批评与指正。

## 核心图像（以最新版本为准）

- 底层不是孤立实体，而是关系；
- 测量不是坍缩，而是观测切面的事实化；
- 客观世界不是全维实在，而是稳定共享的公共投影面；
- 量子纠缠不是远距通信，而是同一联合实在在两处局部记录中的相关显现；
- 宏观空间中的分离，不等于微观关系域中的独立。
